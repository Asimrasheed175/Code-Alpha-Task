# Task 1: Iris Flower Classification
**CodeAlpha Data Science Internship**

## Overview
Classifies Iris flowers (setosa, versicolor, virginica) using sepal and petal measurements.

## Models Used
- K-Nearest Neighbors
- Decision Tree
- Random Forest
- Support Vector Machine ✅ Best (96.67% accuracy)

## Files
- `iris_classification.py` — Main script
- `iris_eda.png` — Exploratory Data Analysis plots
- `iris_corr.png` — Correlation heatmap
- `iris_results.png` — Model comparison & confusion matrix

## Libraries
`scikit-learn`, `pandas`, `numpy`, `matplotlib`, `seaborn`
