# Task 2: Unemployment Analysis with Python
**CodeAlpha Data Science Internship**

## Overview
Analyzes unemployment trends (2019–2021), investigating the impact of COVID-19 on unemployment rates across Urban and Rural regions.

## Key Findings
- COVID-19 spike: +13.6 percentage points at peak
- Urban unemployment consistently higher than rural
- Post-COVID levels remain above pre-COVID baseline

## Files
- `unemployment_analysis.py` — Main analysis script
- `unemployment_analysis.png` — Full visualization dashboard

`pandas`, `numpy`, `matplotlib`, `seaborn`
